Build a production-ready full stack web application called **"Local Service Finder (Hyperlocal Marketplace)"** with the following requirements:

### 🧩 Tech Stack

* Frontend: React.js (with Tailwind CSS)
* Backend: Node.js with Express.js
* Database: MongoDB (with Mongoose)
* Authentication: JWT-based authentication
* Maps & Location: Google Maps API (or OpenStreetMap)
* Real-time features: Socket.io
* AI/ML: Python (FastAPI or Flask microservice)

---

# 🎯 Core Features

## 👤 User Features

* User registration & login (JWT authentication)
* Detect user location (Geolocation API)
* Search for nearby services (plumber, electrician, tutor, etc.)
* Filter by:

  * Distance
  * Rating
  * Price
* Book a service with date & time
* Real-time chat with service provider
* View booking history
* Give ratings & reviews

---

## 🧑‍🔧 Service Provider Features

* Register as a provider
* Add services (category, price, availability)
* Toggle availability (online/offline)
* Accept/reject bookings
* View earnings dashboard

---

## ⚙️ Admin Features

* Manage users and providers
* Remove fake/spam listings
* Monitor platform activity
